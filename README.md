# QuickGen Website

Basic static website (no frameworks) used for demo and documentation.

## Project structure

- `index.html` — Home page
- `about/index.html` — About page
- `assets/css/styles.css` — Styles

## How to view

Open `index.html` in a browser or deploy the repository as a static site (Cloudflare Pages recommended).

## AI Guidelines

- Static site only (Cloudflare Pages)
- No frameworks (no React, no build tools)
- Plain HTML, CSS, JS
- Mobile-first responsive design
- Accessible semantic HTML
